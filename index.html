1.SISITEMA DE CALIFICACIONES DETALLADO

Estructura condicional if() Else if()

Un colegio necesita un sistema para evaluar el rendimiento de los estudiantes en una materia específica. El sistema debe permitir ingresar 10 notas numéricas (entre 0 y 5). Luego, calculará el promedio de estas notas y determinará si el estudiante aprobó o no (se considera aprobado con un promedio de 3.0 o superior).

Análisis:

A continuación se describe el análisis del algoritmo

Datos de entrada:

  • Calificaciones del estuduante

    • Proceso:

    Se suman las notas usando un ciclo for, luego se calcula el promedio. Se utiliza condicional para determinar si el estudiante aprobó o reprobó y para asignar una calificación literal.

    Datos o mensajes de salida:

    Se muestra el promedio calculado con dos decimales, el estado del estudiante (Aprobado o Reprobado) y una calificación literal que puede ser: Reprobado, Aceptable, Sobresaliente o Excelente, según el promedio obtenido.

    Función relizada: 

                    
                   function promedioDeEstudiantes() {
    alert("Este algoritmo permite calcular el promedio de 10 notas ingresadas (entre 0 y 5) para determinar si el estudiante aprueba o reprueba. Además, mostrará una calificación literal basada en el promedio obtenido. ¡Vamos a calcular!")
    let notas = [];
    let suma = 0;
    let promedio;
    let calificacion;

    for (let i = 0; i < 10; i++) {
        let nota = parseFloat(prompt(`Ingrese la nota ${i + 1} (entre 0 y 5):`));
        while (isNaN(nota) || nota < 0 || nota > 5) {
            alert("Nota inválida. Debe estar entre 0 y 5.");
            nota = parseFloat(prompt(`Ingrese nuevamente la nota ${i + 1} (entre 0 y 5):`));
        }
        notas.push(nota);
        suma += nota;
    }

    promedio = suma / 10;

    let estado = promedio >= 3.0 ? "Aprobado" : "Reprobado";

    if (promedio >= 0.0 && promedio <= 2.9) {
        calificacion = "Reprobado";
    } else if (promedio >= 3.0 && promedio <= 3.9) {
        calificacion = "Aceptable";
    } else if (promedio >= 4.0 && promedio <= 4.4) {
        calificacion = "Sobresaliente";
    } else if (promedio >= 4.5 && promedio <= 5.0) {
        calificacion = "Excelente";
    }

    alert(`Promedio: ${promedio.toFixed(2)}\nEstado: ${estado}\nCalificación: ${calificacion}`);
}

    6. CÁLCULO DEL ÍNDICE DE MASA CORPORAL (IMC)

    Estructura condicional if() else if()

    Un sistema de salud permite a los usuarios conocer su Índice de Masa Corporal (IMC) y la categoría en la que se encuentran, ofreciendo además recomendaciones generales:

    Análisis:

    A continuación se describe el análisis del algoritmo

    Datos de entrada:

  • Peso en kilogramos (kg)

  • Estatura en metros (m)

    • Proceso:

    Se solicitan el peso y la estatura del usuario, validando que sean valores numéricos mayores a cero. Luego se calcula el IMC utilizando la fórmula: IMC = peso / (estatura × estatura). Dependiendo del resultado, se clasifica en una de las siguientes categorías: bajo peso, normal, sobrepeso u obesidad. A cada categoría se le asigna una recomendación de salud específica.

    Datos o mensajes de salida:

    • Valor del IMC con dos decimales
    • Categoría correspondiente
    • Recomendación general de salud
    • Mensaje de error si se ingresan datos inválidos

    Función realizada:

      
  function calcularIMC() {
    alert("Este programa te permite calcular tu Índice de Masa Corporal (IMC) ingresando tu peso en kilogramos y tu estatura en metros. Además, el sistema te indicará en qué categoría te encuentras (bajo peso, normal, sobrepeso u obesidad) y te dará recomendaciones generales de salud según el resultado obtenido.");

    // Solicitar y validar el peso
    let peso = parseFloat(prompt("Ingrese su peso en kilogramos (kg):"));
    while (isNaN(peso) || peso <= 0) {
        peso = parseFloat(prompt("Peso inválido. Por favor, ingrese un peso válido en kg:"));
    }

    // Solicitar y validar la estatura
    let estatura = parseFloat(prompt("Ingrese su estatura en metros (m):"));
    while (isNaN(estatura) || estatura <= 0) {
        estatura = parseFloat(prompt("Estatura inválida. Por favor, ingrese una estatura válida en metros:"));
    }

    // Cálculo del IMC
    let imc = peso / (estatura * estatura);
    let categoria = "";
    let recomendacion = "";

    // Clasificación del IMC
    if (imc < 18.5) {
        categoria = "Bajo peso";
        recomendacion = "Se recomienda consultar a un nutricionista.";
    } else if (imc >= 18.5 && imc < 25) {
        categoria = "Normal";
        recomendacion = "¡Mantén un estilo de vida saludable!";
    } else if (imc >= 25 && imc < 30) {
        categoria = "Sobrepeso";
        recomendacion = "Considera realizar más actividad física y mejorar tu dieta.";
    } else {
        categoria = "Obesidad";
        recomendacion = "Es importante buscar asesoramiento médico para un plan de salud integral.";
    }

    // Mostrar resultado
    alert(\`Tu IMC es: \${imc.toFixed(2)}\nCategoría: \${categoria}\nRecomendación: \${recomendacion}\`);
  }

    7. CALCULADORA DE SUELDO CON BONIFICACIONES POR HIJOS

    Estructura condicional if() else if()

    Una empresa calcula el sueldo total de sus empleados considerando un sueldo base y una bonificación adicional basada en el número de hijos. Este sistema permite brindar un apoyo económico adicional a quienes tienen más responsabilidades familiares.

    Análisis:

    A continuación se describe el análisis del algoritmo:

    Datos de entrada:

    • Sueldo base del empleado
    • Número de hijos

    Proceso:

    Se solicita el sueldo base y la cantidad de hijos. Si el empleado no tiene hijos, no recibe bonificación. Si tiene 1 o 2 hijos, recibe un 5% adicional sobre su sueldo base. Si tiene 3 o más, el bono es del 10%. Luego, se suma el bono al sueldo base para obtener el total.

    Datos o mensajes de salida:

    • Valor del bono
    • Sueldo total

    Función realizada:

    
function calcularSueldo() {
  // Mostrar mensaje inicial explicando para qué sirve el programa
  alert("Este programa calcula el sueldo total con base en un sueldo fijo más una bonificación por hijos.");

  // Pedimos al usuario el sueldo base
  let sueldoBase = prompt("Ingrese su sueldo base:");
  // Convertimos a número
  sueldoBase = parseFloat(sueldoBase);

  // Pedimos el número de hijos
  let numHijos = prompt("¿Cuántos hijos tiene?");
  // Convertimos a número
  numHijos = parseInt(numHijos);

  // Creamos una variable para el bono
  let bono = 0;

  // Calculamos el bono según el número de hijos
  if (numHijos == 0) {
    bono = 0;
  } else if (numHijos == 1 || numHijos == 2) {
    bono = sueldoBase * 0.05;
  } else if (numHijos >= 3) {
    bono = sueldoBase * 0.10;
  }

  // Calculamos el sueldo total sumando el bono al sueldo base
  let sueldoTotal = sueldoBase + bono;

  // Mostramos los resultados
  alert("Sueldo base: $" + sueldoBase + "\nBono: $" + bono + "\nSueldo total: $" + sueldoTotal);
}

Luego, se valida que el tipo ingresado sea uno de los válidos (A, B, AB u O). function clasificarSangre() {
    // Mensaje inicial explicando el objetivo del programa
    alert("Este programa ayuda a los donantes y receptores a conocer las compatibilidades sanguíneas según su tipo de sangre. Al ingresar su tipo, el sistema muestra a quién pueden donar y de quién pueden recibir.");

    // Solicita el tipo de sangre y lo convierte a mayúsculas
    let tipoSangre = prompt("Ingrese su tipo de sangre (A, B, AB, O):").toUpperCase();

    let puedeDonar = "";
    let puedeRecibir = "";

    // Validación básica de tipo sanguíneo permitido
    if (tipoSangre !== "A" && tipoSangre !== "B" && tipoSangre !== "AB" && tipoSangre !== "O") {

    // Evaluación de compatibilidad
    switch (tipoSangre) {
        case "A":
            puedeDonar = "A, AB";

        case "B":
            puedeDonar = "B, AB";
            puedeRecibir = "B, O";
            break;
        case "AB":
            puedeDonar = "AB";
            puedeRecibir = "A, B, AB, O";
            break;
        case "O":
            puedeDonar = "A, B, AB, O";
            puedeRecibir = "O";
            break;
    }

    // Mostrar resultado final
    alert("Tipo de Sangre: " + tipoSangre +

    9. CALCULADORA DE NOTAS Y NIVEL DE DESEMPEÑO

    Estructura condicional if() else if()

    Este programa calcula el promedio de tres notas parciales (cada una de 0 a 5) y determina el nivel de desempeño del estudiante con base en el promedio obtenido.

    Análisis:

    Datos de entrada:

    • Tres notas parciales ingresadas por el usuario (números entre 0 y 5).

    Proceso:

    Se solicitan las tres notas y se validan para que estén en el rango correcto. Luego, se calcula el promedio. Según el promedio, se determina el nivel de desempeño y se asigna un mensaje de retroalimentación.

    Datos o mensajes de salida:

    • Promedio de las tres notas.
    • Nivel de desempeño: Bajo, Básico, Alto o Superior.
    • Mensaje con recomendaciones o felicitaciones.

// Función para calcular promedio y nivel de desempeño
function calcularDesempeño() {
  alert("Este programa calcula el promedio de tres notas (0 a 5) y te dice tu nivel de desempeño.");

  // Pedimos la primera nota y validamos que esté entre 0 y 5 y que sea un número
  let nota1 = parseFloat(prompt("Ingresa la primera nota (0 a 5):"));
  while (nota1 < 0 || nota1 > 5 || nota1 != nota1) {
    nota1 = parseFloat(prompt("Valor inválido. Ingresa la primera nota (0 a 5):"));
  }

  // Pedimos la segunda nota con la misma validación
  let nota2 = parseFloat(prompt("Ingresa la segunda nota (0 a 5):"));
  while (nota2 < 0 || nota2 > 5 || nota2 != nota2) {
    nota2 = parseFloat(prompt("Valor inválido. Ingresa la segunda nota (0 a 5):"));
  }

  // Pedimos la tercera nota con la misma validación
  let nota3 = parseFloat(prompt("Ingresa la tercera nota (0 a 5):"));
  while (nota3 < 0 || nota3 > 5 || nota3 != nota3) {
    nota3 = parseFloat(prompt("Valor inválido. Ingresa la tercera nota (0 a 5):"));
  }

  // Calculamos el promedio de las tres notas
  let promedio = (nota1 + nota2 + nota3) / 3;

  // Inicializamos variables para el nivel y el mensaje
  let nivel = "";
  let mensaje = "";

  // Determinamos el nivel según el promedio
  if (promedio < 3) {
    nivel = "Nivel Bajo";
    mensaje = "Necesitas estudiar más para mejorar.";
  } else if (promedio < 4) {
    nivel = "Nivel Básico";

function calcularDescuento() {

  // Pedimos el tipo de cliente
  let tipoCliente = prompt("Ingrese el tipo de cliente (nuevo, frecuente, vip):");
  if (tipoCliente) {
    tipoCliente = tipoCliente.toLowerCase();
  } else {
    alert("Error: Debes ingresar un tipo de cliente.");
    return;
  }

  // Pedimos el valor de la compra
  let valorCompra = parseFloat(prompt("Ingrese el valor total de la compra:"));

        suma = suma + temp;

        // Buscar mayor
        if (temp > mayor) {
            mayor = temp;
        }

        // Buscar menor
        if (temp < menor) {
            menor = temp;
        }

        // Contar días con temperatura mayor a 23
        if (temp > 23) {
            diasCalurosos++;
        }
    }

    // Calcular promedio
    let promedio = suma / temperaturas.length;

    // Mostrar resultados
    alert("Análisis de Temperaturas:\n" +
          "Promedio: " + promedio + "°C\n" +
          "Temperatura más alta: " + mayor + "°C\n" +
          "Temperatura más baja: " + menor + "°C\n" +
          "Días con más de 23 grados: " + diasCalurosos);
}

    • Solicitar los datos de acceso y validar si coinciden.
    • Permitir hasta 3 intentos.

    Datos o mensajes de salida:

    • Mensaje de éxito si los datos coinciden.
    • Mensaje de error en caso de intento fallido.
    • Mensaje de bloqueo tras 3 intentos incorrectos.

                    function registroUsuarios() {
    alert("Este algoritmo permite registrar un usuario con nombre, correo y contraseña. Luego simula el inicio de sesión validando los datos. Si los datos no coinciden, se permite reintentar hasta 3 veces.");

    // Paso 1: Registrar usuario
    let usuario = {
        nombre: prompt("Registre su nombre:"),
        correo: prompt("Registre su correo electrónico:"),
        contrasena: prompt("Cree una contraseña:")
    };

    alert("Registro exitoso. Ahora inicie sesión.");

    // Paso 2: Intentos de acceso

    let accesoConcedido = false;

    while (intentos < 3 && !accesoConcedido) {

            alert("Acceso concedido. Bienvenido, " + usuario.nombre + "!");
            accesoConcedido = true;
        } else {
            intentos++;
            if (intentos < 3) {
                alert("Datos incorrectos. Intente de nuevo. Intento " + intentos + " de 3.");
            }
        }
    }

    if (!accesoConcedido) {
        alert("Acceso bloqueado. Ha superado el número máximo de intentos.");
    }

                function agendaDeContactos() {

    26. SISTEMA DE COMPRAS CON CARRITO

    Ejercicios con Objetos

    El usuario puede agregar productos a un carrito de compras. Cada producto es un objeto con nombre, precio y cantidad. Al finalizar, se muestra el total a pagar y un resumen de los productos.

    Análisis:

    A continuación se describe el análisis del algoritmo:

    Datos de entrada:

    • Nombre del producto.
    • Precio unitario.
    • Cantidad deseada.

    Proceso:

    • Registrar los productos en un arreglo de objetos.
    • Calcular el subtotal por cada producto (precio * cantidad).
    • Sumar todos los subtotales para obtener el total a pagar.
    • Mostrar un resumen con nombre, precio, cantidad y subtotal de cada producto.

    Datos o mensajes de salida:

    • Resumen detallado del carrito.
    • Total general a pagar.

    Función realizada:

                    
                    function sistemaDeCompras() {
    alert("Bienvenido al sistema de compras. Podrás agregar productos con su nombre, precio y cantidad. Al finalizar, verás el resumen del carrito y el total a pagar.");

    let carrito = []; // Arreglo de objetos para almacenar los productos
    let seguir = "si";

    // Paso 1: Agregar productos al carrito
    while (seguir.toLowerCase() === "si") {
        let producto = {
            nombre: prompt("Ingrese el nombre del producto:"),
            precio: parseFloat(prompt("Ingrese el precio unitario del producto:")),
            cantidad: parseInt(prompt("Ingrese la cantidad que desea comprar:"))
        };
        carrito.push(producto); // Se agrega el producto al carrito
        seguir = prompt("¿Deseas agregar otro producto? (si / no)");
    }

    // Paso 2: Calcular total y mostrar resumen
    let total = 0;
    let resumen = "Resumen del carrito:\n";

    for (let i = 0; i < carrito.length; i++) {
        let subtotal = carrito[i].precio * carrito[i].cantidad;
        total += subtotal;

        resumen += "- " + carrito[i].nombre +
            " | Precio: $" + carrito[i].precio.toFixed(2) +
            " | Cantidad: " + carrito[i].cantidad +
            " | Subtotal: $" + subtotal.toFixed(2) + "\n";
    }

    resumen += "\Total a pagar: $" + total.toFixed(2);
    alert(resumen);
}
